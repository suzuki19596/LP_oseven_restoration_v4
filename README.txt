【納品内容】
このフォルダには以下のファイルが含まれています:

∟ index.html          ... ランディングページのHTMLファイル
∟ 📁css/
　　∟ style.css      ... スタイルシート（デザイン設定）
∟ 📁images/
　　∟ (各種画像ファイル) ... ロゴ、施工事例画像、地図など
∟ README.txt          ... 本ファイル（納品物の説明）
∟ 設定手順.pdf        ... 公開手順とGoogle設定方法

【ファイル構成の説明】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. index.html
   - ランディングページのメインファイルです
   - ブラウザでこのファイルを開くとページが表示されます
   - 画像やCSSファイルとの連携が設定済みです

2. css/style.css
   - ページのデザイン（色、レイアウト、アニメーションなど）を
     制御するスタイルシートです
   - HTMLファイルから自動的に読み込まれます

3. images/
   - ページで使用する全ての画像が格納されています
   - ロゴ、施工事例のBefore/After画像、エリアマップなど

4. 設定手順.pdf
   - サーバーへのアップロード方法
   - Google Search Console への登録方法
   - Google Analytics の設定方法
   などが記載されています

【使用方法】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ローカルでの確認
1. index.html をダブルクリック
2. ブラウザで開いて内容を確認できます

■サーバーへのアップロード
詳細は「設定手順.pdf」をご参照ください。

基本的な手順:
1. FTPソフト（FileZilla等）でサーバーに接続
2. このフォルダ内の全てのファイルをアップロード
3. ファイル構造を保ったままアップロードしてください
   （css/やimages/のフォルダごと）

【注意事項】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

※ フォルダ構造を変更しないでください
   （css/, images/ の位置を変えるとページが正しく表示されません）

※ ファイル名を変更しないでください
   （リンクが切れる原因となります）

※ 画像を差し替える場合は、同じファイル名で上書きしてください
