　ランディングページ　設定手順
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

このファイルは、設定手順.pdf に含めるべき内容の説明です。
PDFは手動で作成する必要があります。

【1. サーバーへのアップロード手順】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■必要なもの
- FTPソフト（FileZilla、WinSCP など）
- サーバーのFTP接続情報
  ・ホスト名（FTPサーバー）
  ・ユーザー名
  ・パスワード
  ・ポート番号（通常は21）

■アップロード手順
1. FTPソフトを起動
2. サーバーに接続
3. アップロード先のディレクトリに移動
   （例: /public_html/ または /www/ など）
4. 以下のファイル・フォルダを全てアップロード:
   ・index.html
   ・css フォルダ（中身も含めて）
   ・images フォルダ（中身も含めて）
5. アップロード完了後、ブラウザでURLにアクセスして確認

■確認ポイント
✓ 画像が正しく表示されているか
✓ デザイン（色、レイアウト）が崩れていないか
✓ リンクが正しく動作しているか
✓ スマートフォンでも正しく表示されるか

【2. Google Search Console 設定手順】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■Google Search Console とは
検索結果でのサイトのパフォーマンスを監視・管理できる
Googleの無料サービスです。

■登録手順
1. https://search.google.com/search-console/ にアクセス
2. Googleアカウントでログイン
3. 「プロパティを追加」をクリック
4. サイトのURLを入力（例: https://www.o-seven.co.jp）
5. 所有権の確認方法を選択:
   【おすすめ】HTMLファイルをアップロード
   - 提供されたHTMLファイルをダウンロード
   - FTPでサーバーのルートディレクトリにアップロード
   - 「確認」ボタンをクリック

6. サイトマップの送信（オプションだが推奨）
   - 左メニューから「サイトマップ」を選択
   - サイトマップのURLを入力して送信

■確認できる情報
- 検索キーワード
- クリック数、表示回数
- サイトへの訪問者数
- モバイルユーザビリティの問題
- セキュリティの問題

【3. Google Analytics 設定手順】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■Google Analytics とは
サイトへのアクセス数や訪問者の行動を詳しく分析できる
Googleの無料サービスです。

■登録手順（GA4版）
1. https://analytics.google.com/ にアクセス
2. Googleアカウントでログイン
3. 「測定を開始」をクリック
4. アカウント名を入力（例: O-SEVEN）
5. プロパティの設定:
   - プロパティ名: 「O-SEVEN LP」など
   - タイムゾーン: 日本
   - 通貨: 日本円（JPY）
6. ビジネスの詳細を入力
7. 「ウェブ」を選択
8. ウェブサイトのURLとストリーム名を入力
9. 測定IDをコピー（G-XXXXXXXXXX の形式）

■HTMLファイルへの設定
10. index.html を開く
11. <head> タグ内に以下のコードを追加:

<!-- Google tag (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>

※ G-XXXXXXXXXX は実際の測定IDに置き換えてください

12. 修正したindex.htmlを再度サーバーにアップロード
13. リアルタイムレポートで動作確認

■確認できる情報
- リアルタイムのアクセス数
- ユーザー数、セッション数
- ページビュー数
- 平均セッション時間
- 流入元（どこからサイトに来たか）
- デバイス別の訪問者数
- 地域別の訪問者数

【4. その他の推奨設定】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ファビコンの設定
ブラウザのタブに表示されるアイコンです。
1. 16x16px または 32x32px のアイコンを用意
2. ファイル名を「favicon.ico」として保存
3. サーバーのルートディレクトリにアップロード

■OGP設定（SNSシェア用）
TwitterやFacebookでシェアされた時の見え方を設定します。
index.htmlの<head>内に以下を追加:

<meta property="og:title" content="株式会社O-SEVEN - 北摂・豊中の原状回復専門">
<meta property="og:description" content="高品質 × 適正価格 × 迅速対応">
<meta property="og:image" content="https://www.o-seven.co.jp/images/ogp.jpg">
<meta property="og:url" content="https://www.o-seven.co.jp">
<meta name="twitter:card" content="summary_large_image">

■SSL証明書の設定
サーバーでSSL証明書を設定し、https:// でアクセスできるようにします。
（多くのレンタルサーバーでは無料SSL証明書を提供しています）

【5. トラブルシューティング】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■画像が表示されない場合
→ images フォルダが正しくアップロードされているか確認
→ ファイル名の大文字・小文字が一致しているか確認

■デザインが崩れている場合
→ css フォルダが正しくアップロードされているか確認
→ ブラウザのキャッシュをクリアして再読み込み

■スマホで表示が崩れる場合
→ ブラウザの開発者ツールでレスポンシブ表示を確認
→ 必要に応じてCSS調整
